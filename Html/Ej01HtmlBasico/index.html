Cuerpo pricipal del contenido

La etiqueta main es en HTML5 una etiqueta semántica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la página. Unicamente solo puede haber una sola etiqueta como esta en todo el documento.
El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de página, los menues de navegación y los pie de página). Es usado por los navegadores para indexar.
Conviene usar aquí textos que ayuden a encontrar la página luego de procesos de indexación de los buscadores.

Sección etiquetas semánticas estructurales más importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRES

Las etiquetas semánticas estructurales están relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento

Reglas para etiquetas semánticas

  • Los distintos tipos de contenido se pueden asociar a las etiquetes SECTION
  • Las unidades de cintenido se asocian a las etiquetes ARTICLE. O sea que en un SECTION anida multiples ARTICLES
  • Las etiquetes HEADER y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. O sea un FOOTER podria anidar otro HEADER y FOOTER
  • La etiqueta NAV debe ser única en el documento y corresponde a los elementos de navegación del documento
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main, header o footer

Respuestas a las preguntas de HTML

  1. Ejemplos de uso de etiquetes meta son:
    La etiqueta "description" se utiliza para proporcionar una descripción breve del contenido de la página, que los motores de búsqueda pueden utilizar para mostrar en los resultados de búsqueda.
    La etiqueta "viewport" se utiliza para indicar al navegador que la página debe ser responsive y ajustarse al ancho de la pantalla del dispositivo en el que se está visualizando.

  2. Algunos atributos aplicables a las etiquetas HTML son:
    El atributo "href" se utiliza en la etiqueta para indicar la URL del enlace.
    El atributo "src" se utiliza en la etiqueta para indicar la URL de la imagen que se debe mostrar.

  3. La etiqueta "head" se relaciona con el header HTTP de modo que el programador, al no tener el poder para controlar los encabezados que tendra el header HTTP, usa la etiqueta head para poder darle cirta información adicional que quiera pasar.

  4. UTF-8 es una codificación de caracteres que se utiliza para representar caracteres de texto en formato electrónico. Permite representar caracteres de múltiples idiomas y tiene una capacidad de memoria de 4 bytes, a diferencia de la codificación ASCII que solo puede contener 1 byte. Esta cantidad de memoria lo hace compatible con la mayoría de los sistemas informáticos y navegadores web debido a que puede representar mayor cantidad de símbolos.

  5. El atributo HTML que se utiliza para aplicar estilo es el atributo "style", que se puede aplicar a cualquier etiqueta HTML.

  6. Para indicar que un enlace debe abrirse en una nueva pestaña del navegador, se puede utilizar el atributo "target" con el valor "_blank" en la etiqueta "a". Por ejemplo, "a href="ejemplo" target="_blank".

  7. Las etiquetas semánticas en HTML se utilizan para dar significado al contenido de la página. Por ejemplo, se puede utilizar "header" para indicar que el contenido que está dentro de esa etiqueta es el encabezado de la página, o "section" para indicar que el contenido que está dentro de esa etiqueta es una sección independiente. El propósito de las etiquetas semánticas es mejorar la accesibilidad y la legibilidad del contenido para los motores de búsqueda y otros sistemas automatizados.

  8. Algunos atributos de estilo básicos que se pueden aplicar a una tabla en HTML son:
    "border": indica el ancho del borde de la tabla.
    "padding": indica el espacio entre el contenido de las celdas y el borde de la tabla.
    "background-color": indica el color de fondo de la tabla.

  9. La etiqueta "body" en HTML se utiliza para indicar el contenido visible de la página web. El cuerpo del protocolo HTTP, por su parte, es la parte de una solicitud o respuesta HTTP que contiene el contenido real que se está enviando o recibiendo.

  10. Los atributos que se utilizan para agrupar celdas en una tabla son "colspan" y "rowspan". Ambos atributos se aplican a las etiquetas "td" y "th", que son las etiquetas que se utilizan para definir las celdas en una tabla HTML.
